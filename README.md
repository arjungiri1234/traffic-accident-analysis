# Traffic-Accident-Analysis

## Milestones 2 - Data Cleaning and EDA notebook


## Key Findings
- California, Florida, and Texas have the highest accident counts
- Most accidents occur during rush hours (7–9 AM and 4–6 PM)
- Severity Level 2 accounts for the majority of accidents
- Fair/Clear weather has the most accidents overall
- Traffic signals are present at a high proportion of accident locations

## How to Run
1. Clone the repository
2. Download the dataset from Kaggle and place in the project folder
3. Install dependencies:
```bash
   pip install pandas numpy matplotlib seaborn
```
4. Open and run `traffic_eda_notebook.ipynb` in VS Code or Jupyter
